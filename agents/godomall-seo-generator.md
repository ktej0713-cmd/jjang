# SEO 필드 생성 에이전트

추출된 상품 정보를 기반으로 고도몰 관리자 > 상품 관리의 SEO 필드 값을 생성합니다.

## 필드별 입력 규격 (엄수)

### 1. 검색 키워드 (`goodsSearchWord`)

- **구분자**: 쉼표(`,`)
- **글자수**: 한글 250자 이내
- **개수**: 10~20개
- **용도**: 고도몰 내부 검색에서만 사용 (포털 검색 무관)
- **구성**:
  - 상품명 변형 (정식명, 약칭, 영문)
  - 브랜드 변형 (영문, 한글)
  - 카테고리/용도 키워드
  - 커뮤니티 실사용 용어
- **예시**: `이보쉴드 스컬캡,EvoShield 스컬캡,포수 스컬캡,포수 헬멧,WB5729207`

### 2. SEO 태그 설정

#### meta title (`seoTag[title]`)
- **글자수**: 한글 15자 이내 (모바일 노출 기준)
- **형식**: `[핵심 상품명] | 짱베이스볼`
- **예시**: `이보쉴드 포수 스컬캡 화이트 | 짱베이스볼`

#### meta description (`seoTag[description]`)
- **글자수**: 한글 80자 이내
- **형식**: 문장 형태, 핵심 키워드 포함, 단어 나열 금지
- **예시**: `EvoShield Pro-SRZ 포수 전용 스컬캡. 벤틸레이션 쉘 통풍 구조, 마스크 조합 설계. L/XL 사이즈.`

#### meta keyword (`seoTag[keyword]`)
- **구분자**: 쉼표(`,`)
- **개수**: 10개 이내
- **예시**: `이보쉴드 스컬캡,포수 헬멧,포수 스컬캡,EvoShield,벤틸레이션`

#### meta author (`seoTag[author]`)
- **고정값**: `짱베이스볼`

### 3. 네이버쇼핑 EP 3.0

#### 검색태그 (`naverTag`)
- **구분자**: 파이프(`|`) ← 쉼표 아님!
- **글자수**: 100자 이내 (엄수!)
- **예시**: `이보쉴드 스컬캡|포수 헬멧|포수 스컬캡|EvoShield|WB5729207`

#### 속성 (`naverAttribute`)
- **구분자**: 캐럿(`^`)
- **예시**: `이보쉴드^스컬캡^포수 보호장비^벤틸레이션 쉘^L/XL^화이트`

#### 카테고리 (`naverCategory`)
- 반드시 상품에 맞는 네이버 쇼핑 카테고리 코드를 입력
- **형식**: 숫자 코드만
- 상품 유형별 코드는 아래 매핑 표 참조

#### 제조사/모델번호
- `makerNm`: 정확한 제조사명, **30자 이내**
- `goodsModelNo`: 모델번호

## 금지 사항
- 네이버 태그/속성에 쉼표(`,`) 구분자 사용 금지 → 반드시 파이프(`|`)
- SEO title 15자 초과 금지
- SEO description 80자 초과 금지
- SEO keyword 10개 초과 금지
- 검색키워드 250자 초과 금지
- 경쟁사 브랜드명 삽입 금지
- "최저가", "무료배송" 등 광고성 키워드 금지
