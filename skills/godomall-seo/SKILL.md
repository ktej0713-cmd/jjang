# 고도몰 상품 SEO 자동 입력 스킬

## 목적
고도몰5 관리자 > 상품 관리에서 입력해야 하는 SEO 관련 3개 영역의 값을 자동 생성합니다.
관리자 페이지에 바로 복사-붙여넣기할 수 있는 형식으로 출력합니다.

## 에이전트 팀

이 스킬은 4개의 전용 에이전트가 파이프라인으로 동작합니다:

```
고도몰 SEO 리드 (오케스트레이터)
├── 상품정보 추출기 — HTML/데이터에서 상품 정보 추출
├── SEO 필드 생성기 — 3개 영역 값 생성
└── SEO 검증기 — 규격/정책 검증
```

## 대상 필드 (고도몰 관리자 > 상품 관리)

### [필드1] 검색 키워드
- **위치**: 상품 등록/수정 > 기본 정보 > 검색 키워드
- **형식**: 쉼표(,) 구분 키워드 15~25개
- **목적**: 고도몰 내부 검색 노출

### [필드2] 상품 개별 SEO 태그 설정
- **위치**: 상품 등록/수정 > SEO > 상품 개별 SEO 태그 설정
- **필드 목록**:
  - meta title (한글 15자 이내 권장, 모바일 노출 기준)
  - meta description (한글 80자 이내 권장, 문장 형태)
  - meta keywords (쉼표 구분, 10개 이내)
  - meta author (고정: 짱베이스볼)

### [필드3] 네이버쇼핑 EP 3.0 정보 등록
- **위치**: 상품 등록/수정 > 네이버쇼핑EP
- **필드 목록**:
  - 네이버 카테고리 코드 (숫자)
  - 제조사 (`makerNm`)
  - 모델번호 (`goodsModelNo`)
  - 검색태그 (`naverTag`) -- 파이프(`|`) 구분! 쉼표 아님!
  - 속성 (`naverAttribute`) -- 파이프(`|`) 구분! 쉼표 아님!

## 실행 흐름

### 모드 1: 상세페이지 HTML 기반 (자동 추출)
```
1. 사용자가 HTML 파일 지정 또는 IDE에서 선택
2. 상품정보 추출기가 HTML에서 정보 추출
3. 누락 필드 확인 → 사용자에게 보완 요청
4. SEO 필드 생성기가 3개 영역 값 생성
5. SEO 검증기가 검증
6. 관리자 입력용 출력
```

### 모드 2: 직접 입력
```
1. 사용자가 상품 정보 직접 제공
2. SEO 필드 생성기가 3개 영역 값 생성
3. SEO 검증기가 검증
4. 관리자 입력용 출력
```

### 모드 3: Open API 자동 입력 (가능한 필드만)
```
1. 모드 1 또는 2로 값 생성
2. 고도몰 Open API (Goods_Update) 호출하여 지원 필드 자동 입력
3. API 미지원 필드는 복사-붙여넣기 안내
```

## 최소 필수 정보

| 항목 | 필수 | 비고 |
|------|------|------|
| 상품명 | O | 한글 + 영문 |
| 브랜드 | O | 한글 + 영문 |
| 카테고리 | O | 대분류 > 중분류 > 소분류 |
| 상품 설명 (1~2줄) | O | 체감 중심 |
| 가격 | - | EP 3.0에 필요 |
| 모델번호 | - | EP 3.0에 필요 |
| 색상 | - | 속성 정보 |
| 사이즈 | - | 속성 정보 |
| 이미지 URL | - | OG image |

## 키워드 생성 규칙

### 검색 키워드 구성 (15~25개)
- 상품명 변형 (30%): 정식명, 약칭, 영문, 한글
- 브랜드 변형 (20%): 영문, 한글, 약칭
- 카테고리/용도 (20%): 카테고리명, 용도
- 타겟 (15%): 사회인야구, 유소년야구, 엘리트야구
- 커뮤니티 용어 (15%): 엠엘비파크/디시 등 실사용 표현

### 금지 키워드
- 경쟁사 브랜드명
- "최저가", "무료배송", "파격할인", "특가", "1위", "베스트"
- 관련 없는 인기 키워드

## JSON-LD 규칙
- Product + Offer + BreadcrumbList 조합
- FAQPage 스키마 사용 금지 (2023.08 이후 제한)
- HowTo 스키마 사용 금지 (2023.09 deprecated)
- 가격은 숫자만 (콤마/원 기호 제외)
- URL은 절대 경로
