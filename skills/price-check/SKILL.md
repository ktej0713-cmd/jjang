# 가격 체크

특정 상품의 시장 가격을 조사하여 경쟁 포지셔닝을 파악합니다.

## 워크플로우

### Step 1: 상품 정보 파싱

`$ARGUMENTS`에서 상품명, 브랜드, 모델명을 추출합니다.

### Step 2: 가격 조사 (병렬 검색)

다음 채널에서 가격을 조사합니다:

**네이버 쇼핑:**
```
WebSearch: "상품명 사이트:shopping.naver.com"
```

**주요 야구용품몰:**
```
WebSearch: "상품명 site:yaguyacom.co.kr OR site:baseballpark.co.kr OR site:jjangbaseball.com"
```

**쿠팡/11번가:**
```
WebSearch: "상품명 site:coupang.com OR site:11st.co.kr"
```

### Step 3: 가격 비교표 생성

```markdown
## 가격 비교 결과

**조사 상품:** [상품명]
**조사일:** YYYY-MM-DD

### 채널별 가격
| 채널 | 가격 | 배송비 | 실구매가 | 비고 |
|------|------|--------|---------|------|
| 짱베이스볼 | - | - | - | 현재 미등록/등록가 |
| 네이버 최저가 | - | - | - | |
| 야구야닷컴 | - | - | - | |
| 베이스볼파크 | - | - | - | |
| 쿠팡 | - | - | - | |

### 가격 분석
- **시장 평균가:**
- **최저가:**
- **최고가:**
- **권장 판매가:** (마진율 고려)

### MD 의견
- 가격 경쟁력 평가
- 추천 가격대
- 프로모션 가격 제안 (해당 시)
```

### Step 4: 브랜드 카탈로그 참조

`.claude/knowledge/brand-catalog.md`에 해당 브랜드 정보가 있으면 참조하여:
- 공급가/마진율 정보 확인
- 기존 가격 정책과 비교
- 시즌별 가격 변동 패턴 확인

## 주의사항

- 가격은 조사 시점 기준이며 실시간 변동 가능
- 웹 검색 기반이므로 정확도 한계가 있음을 명시
- 최종 가격 결정은 반드시 실제 공급가/마진율 확인 후 진행
- 과장된 "최저가" 표현은 사용하지 않음
