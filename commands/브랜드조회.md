# 브랜드 조회

짱베이스볼 취급 브랜드의 상세 정보와 라인업을 조회하는 커맨드입니다.

## 입력
브랜드명을 제공합니다. 예시:
- `/브랜드조회 미즈노`
- `/브랜드조회 윌슨`
- `/브랜드조회 구보타슬러거`

인자: $ARGUMENTS

## 실행 절차

### 1단계: 브랜드 데이터 로드
- `~/.claude/knowledge/brand-catalog.md`에서 해당 브랜드 정보 추출
- `~/.claude/knowledge/price-range-guide.md`에서 가격 포지션 확인
- `~/.claude/knowledge/category-structure.md`에서 취급 카테고리 확인

### 2단계: 브랜드 프로필 출력

#### 기본 정보
| 항목 | 내용 |
|------|------|
| 브랜드명 | 한글명 (영문명) |
| 국가 | 원산국 |
| 취급 카테고리 | 글러브/배트/보호장비/야구화/의류 |
| 가격대 | 최저~최고 |
| 주요 타겟 | 사회인/유소년/엘리트 |
| 포지셔닝 | 입문/중급/프리미엄/최고급 |

#### 주력 라인업
| 시리즈/모델 | 카테고리 | 가격 | 특징 | 추천 타겟 |
|------------|---------|------|------|----------|

#### 경쟁 브랜드 비교
- 같은 가격대 경쟁 브랜드와 비교
- 차별화 포인트
- 짱베이스볼에서의 판매 포지션

#### MD 코멘트
- 이 브랜드를 추천하는 상황
- 주의할 점 (사이즈, 관리법 등)
- 커뮤니티 평가/인지도

### 3단계: 관련 콘텐츠 제안
- 이 브랜드로 만들 수 있는 기획전 아이디어
- 상세페이지 작성 시 강조 포인트
- 네이버 쇼핑 키워드 제안

## 출력 원칙
- 야구를 아는 MD가 브리핑하는 톤
- 브랜드 과대평가/과소평가 없이 객관적
- 실제 판매에 활용 가능한 정보 위주
