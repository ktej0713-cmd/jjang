# 상세페이지 HTML 생성

짱베이스볼 상품 상세페이지를 표준 구조로 생성하는 커맨드입니다.
GEO/AEO 최적화가 기본 포함됩니다.

## 입력
상품명 또는 URL을 제공합니다. 예시:
- `/상세페이지 https://www.underarmour.co.kr/...`
- `/상세페이지 미즈노 프로 BSS 글러브`
- `/상세페이지 윌슨 A2000 CK22`

인자: $ARGUMENTS

## 실행 절차

### 1단계: 상품 정보 수집
- URL이 제공된 경우 WebFetch로 상품 정보 스크래핑
- `~/.claude/knowledge/brand-catalog.md`에서 브랜드/가격대 확인
- `~/.claude/knowledge/price-range-guide.md`에서 포지셔닝 확인
- 상품 카테고리 자동 판별 (글러브/배트/보호장비/야구화/의류/용품)

### 2단계: 템플릿 선택
- `~/.claude/knowledge/templates/` 에서 카테고리 맞는 템플릿 로드
  - 글러브: glove-detail-template.html
  - 배트: bat-detail-template.html
  - 보호장비/기타: protection-detail-template.html
- 템플릿이 없는 카테고리는 보호장비 템플릿(범용) 기반으로 생성

### 3단계: 8단계 표준 구조 작성
1. 히어로: 상품 이미지 + 핵심 카피 1줄
2. 필요 상황: 이 상품이 필요한 구체적 상황 (공감 유도)
3. 핵심 스펙: 주요 사양 표 (웹/무게/사이즈/소재)
4. 체감 설명: 실사용자 관점 설명 (스펙 나열 X, 체감 O)
5. 사이즈 가이드: 포지션별/학년별 추천
6. 비교: 같은 카테고리 경쟁 상품과 비교 표
7. 추천 포인트: 누구에게 추천하는지 명확히
※ 배송/교환/반품 안내는 고도몰 기본 기능으로 별도 제공 — 상세페이지에 포함하지 않음

### 4단계: GEO/AEO 최적화 (필수 — 모든 상세페이지에 자동 포함)

#### 4-1. JSON-LD Product 스키마 (`<head>` 내 삽입)
```json
{
  "@context": "https://schema.org/",
  "@type": "Product",
  "name": "상품명",
  "alternateName": "영문 상품명",
  "description": "150자 이내 핵심 설명",
  "brand": { "@type": "Brand", "name": "브랜드명" },
  "sku": "상품코드",
  "material": "소재",
  "color": "색상",
  "size": "사이즈 옵션",
  "offers": {
    "@type": "Offer",
    "url": "https://jjangbaseball.com",
    "priceCurrency": "KRW",
    "price": "가격(숫자만)",
    "availability": "https://schema.org/InStock",
    "seller": { "@type": "Organization", "name": "짱베이스볼" }
  }
}
```

#### 4-2. JSON-LD FAQPage 스키마 (`<head>` 내 삽입)
- 상품 관련 실구매자가 자주 묻는 질문 5개
- 형식: `{"@type": "Question", "name": "질문?", "acceptedAnswer": {"@type": "Answer", "text": "답변"}}`
- 질문 유형: 사이즈 선택 / 소재 / 가격 / 착용법 / 규정 허용 여부 등

#### 4-3. 상품 정의 문장 블록 (히어로 바로 아래 삽입)
- `"[상품명]은 [핵심기능]을 갖춘 [카테고리]입니다."` 형식의 두괄식 정의
- AI 검색 엔진이 바로 인용할 수 있는 명확한 1~3문장
- 가격·사이즈·소재를 한 문단에 포함

### 5단계: HTML 생성 (2개 파일 필수 출력)
- `~/.claude/output/상세페이지/` 폴더에 아래 2개 파일 동시 생성

#### 파일 1: 미리보기용 (전체 HTML)
- 파일명: `상세_{브랜드}_{상품명}_{날짜}.html`
- 완전한 `<!DOCTYPE html>` 구조 (head/body 포함)
- 브라우저에서 바로 열어 확인 가능

#### 파일 2: 고도몰 붙여넣기용
- 파일명: `상세_{브랜드}_{상품명}_{날짜}_고도몰.html`
- `<html>/<head>/<body>` 태그 없이 콘텐츠만
- CSS 클래스를 `#jjang-detail` ID로 스코핑 (고도몰 기본 스타일 충돌 방지)
- JS 함수명에 `jjang` 접두사 (전역 함수명 충돌 방지)
- JSON-LD 스크립트 상단 포함
- Noto Sans KR 폰트 link 태그 포함
- **고도몰 상품 상세설명 > HTML 소스 모드에 전체 붙여넣기**

공통:
- 모바일 우선 반응형 (860px 기준)
- 이미지: 실제 URL 사용 또는 플레이스홀더 포함

### 6단계: 검증
- 야구 커뮤니티에서 신뢰를 잃을 표현 없는지 확인
- 스펙 정확성 확인
- 타겟(사회인/유소년/엘리트) 적합성 확인
- GEO/AEO 요소 3종 포함 여부 확인 (Product스키마 / FAQPage / 정의문장)

## 출력 원칙
- 복사해서 바로 고도몰에 붙여넣기 가능한 완성도
- 야구를 아는 사람이 쓴 글이 느껴져야 함
- 과장 없는 실사용 기반 설명
- GEO/AEO: 두괄식 + 불릿 + 표 + FAQ 구조 필수
- 네이버 쇼핑 1순위 최적화 — 상품명에 검색 키워드 자연스럽게 포함
