# 상세페이지 HTML 생성

짱베이스볼 상품 상세페이지를 고도몰 붙여넣기용 HTML로 생성합니다.

인자: $ARGUMENTS

---

## 실행 절차 (3단계)

### 1단계: 상품 정보 수집

URL이 제공된 경우 WebFetch로 스크래핑한다. 상품명만 제공된 경우 아래 내장 지식을 사용한다.

**카테고리 판별 기준:**
- 글러브/미트 → 글러브
- 배트 → 배트
- 헬멧/암가드/풋가드/마스크/프로텍터/렉가드 → 보호장비
- 스파이크/인조잔디화/포인트화 → 야구화
- 바지/상의/점퍼/모자/양말 → 의류
- 배팅장갑/수비장갑 → 장갑
- 나머지 → 용품(범용)

### 2단계: HTML 생성

아래 구조와 규칙으로 즉시 HTML을 생성한다. 추가 파일 읽기 없이 이 커맨드의 내용만으로 완성한다.

#### 10단계 표준 구조

1. **히어로** — 상품 이미지 플레이스홀더 + 핵심 카피 1줄 + 상품 정의 문장 (두괄식, AI 검색 최적화)
2. **필요 상황** — "이런 분께 맞습니다" 공감 유도 (3~4가지 구체적 상황)
3. **핵심 스펙** — 주요 사양 표 (소재/무게/사이즈/색상/규격 등)
4. **체감 설명** — 실사용자 관점 설명 (스펙 나열 금지, 손에 잡히는 체감 위주)
5. **포지션별 추천** — 포지션/타겟(사회인/유소년/엘리트)별 적합도
6. **사이즈 가이드** — 사이즈 선택 기준, 호수별 추천 (글러브/배트/야구화에 필수)
7. **비교표** — 같은 카테고리 경쟁 상품 2~3개와 비교
8. **FAQ** — 실구매자 자주 묻는 질문 5개 (아코디언 UI)
9. **추천 포인트** — "이런 분께 추천합니다" 명확한 대상 정리
10. **배송 안내** — 고도몰 기본 기능 안내 문구 (짧게)

#### 디자인 규칙

- 폰트: `'Noto Sans KR', -apple-system, BlinkMacSystemFont, sans-serif` (Google Fonts CDN 포함)
- 컬러: `--primary: #1B2A4A`, `--accent: #D4A843`
- 섹션 제목: `border-left: 4px solid #D4A843` + `padding-left: 16px`
- 이모지 사용 금지
- 최대 너비 860px (고도몰/네이버 스마트스토어 호환)
- 모바일 반응형 (768px 기준)
- 이미지: `860×500px` 회색 플레이스홀더 (`background:#eee; text-align:center`)
- 과장 표현 금지 ("최저가", "파격 할인" 등)

#### GEO/AEO 최적화 (필수 포함)

**JSON-LD Product 스키마** (`<script type="application/ld+json">` — 상단 포함):
```
@type: Product, name, alternateName, description(150자), brand, sku, material, color, size
offers: @type Offer, priceCurrency KRW, price(숫자), availability InStock, seller 짱베이스볼
```

**JSON-LD FAQPage 스키마** (별도 스크립트 — 5개 질문):
- 사이즈 선택 / 소재 / 착용법 / 규정 허용 여부 / 관리법 등 실구매자 질문

**상품 정의 문장** (히어로 바로 아래):
- `"[상품명]은 [핵심기능]을 갖춘 [카테고리]입니다."` — 두괄식 1~3문장

### 3단계: 파일 저장

`C:\Users\NO\.claude\output\상세페이지\` 폴더에 **1개 파일만** 저장한다.

**파일명**: `상세_{브랜드}_{상품명}_{날짜}_고도몰.html`
- 날짜 형식: YYYYMMDD (예: 20260312)
- 공백 제거, 특수문자 없이

**파일 구조** (고도몰 HTML 소스 모드에 직접 붙여넣기용):
```
<link> (Noto Sans KR Google Fonts)
<script type="application/ld+json"> (Product 스키마)
<script type="application/ld+json"> (FAQPage 스키마)
<style> (CSS — #jjang-detail 스코핑)
<div id="jjang-detail"> (전체 콘텐츠)
<script> (아코디언 JS — jjang 접두사)
```

`<html>/<head>/<body>` 태그 없음. CSS와 JS 함수는 모두 `#jjang-detail` 또는 `jjang` 접두사로 스코핑해 고도몰 기본 스타일 충돌 방지.
