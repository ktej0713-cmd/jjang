# 품절체크

짱베이스볼 자사몰 + 경쟁 야구몰의 품절/재고 상태를 체크하고 리포트를 생성합니다.

## 입력
카테고리, 브랜드, 또는 "전체"를 지정합니다. 예시:
- `/품절체크` — 주요 상품 전체 체크
- `/품절체크 글러브` — 글러브 카테고리 집중
- `/품절체크 미즈노` — 미즈노 브랜드 전체
- `/품절체크 배트 경쟁몰` — 배트 카테고리 경쟁몰 비교

인자: $ARGUMENTS

## 실행 절차

### 1단계: 내부 데이터 로드
- `~/.claude/knowledge/brand-catalog.md` — 취급 브랜드/상품 목록
- `~/.claude/knowledge/competitor-tracking.md` — 경쟁몰 정보 및 모니터링 기준 상품
- `~/.claude/knowledge/seasonal-calendar.md` — 현재 시즌 핵심 상품 확인

### 2단계: 자사몰 품절 체크
짱베이스볼(jjangbaseball.com) 사이트에서 품절 상태 확인:

1. WebFetch로 카테고리 페이지 접근
2. 품절 표시 감지: "품절", "일시품절", "soldout", "sold-out" 클래스/텍스트
3. 카테고리별 품절 현황 집계

체크 대상 카테고리:
| 카테고리 | URL 패턴 | 우선순위 |
|---------|---------|---------|
| 글러브 | /goods/goods_list.php?cateCd=글러브카테고리 | 최우선 (핵심 카테고리) |
| 배트 | /goods/goods_list.php?cateCd=배트카테고리 | 높음 |
| 보호장비 | /goods/goods_list.php?cateCd=보호장비카테고리 | 높음 |
| 야구화 | /goods/goods_list.php?cateCd=야구화카테고리 | 중간 |
| 의류 | /goods/goods_list.php?cateCd=의류카테고리 | 중간 |
| 장갑 | /goods/goods_list.php?cateCd=장갑카테고리 | 보통 |

### 3단계: 경쟁몰 재고 비교
자사몰 품절 상품을 경쟁몰에서 검색:

| 사이트 | URL | 체크 방법 |
|--------|-----|----------|
| 야구야 | yaguya.co.kr | 상품 검색 → 재고 상태 확인 |
| 야구넷 | yagooonet.com | 상품 검색 → 재고 상태 확인 |
| 베이스볼파크 | ballpark.co.kr | 상품 검색 → 재고 상태 확인 |
| 네이버쇼핑 | search.shopping.naver.com | 상품명 검색 → 판매처/가격/재고 확인 |

비교 분류:
- **우리만 품절** (기회 손실) → 긴급 재입고 필요
- **경쟁몰만 품절** (판매 기회) → 마케팅 강화, 가격 유지/인상 검토
- **양쪽 다 품절** (공급 이슈) → 브랜드사/도매처 확인 필요
- **양쪽 다 재고** (정상) → 가격 경쟁력만 모니터링

### 4단계: 리포트 생성

출력 파일: `~/.claude/output/품절체크/품절리포트_{YYYYMMDD}.html`

리포트 구조:
```
1. 요약 대시보드
   - 전체 품절률 (카테고리별)
   - 긴급 알림 수
   - 전주 대비 변동

2. 긴급 항목 (빨간색 표시)
   - 베스트셀러 품절 목록
   - 우리만 품절인 상품 (경쟁몰에는 재고 있음)
   - 시즌 핵심 상품 품절

3. 카테고리별 상세
   - 글러브: 브랜드별 품절 현황
   - 배트: 종류별 품절 현황
   - 보호장비/야구화/의류/장갑

4. 경쟁몰 비교표
   | 상품명 | 짱베이스볼 | 야구야 | 야구넷 | 베이스볼파크 | 네이버 |
   |--------|----------|--------|--------|-----------|--------|
   | (상품) | 품절/재고 | 재고/가격 | ... | ... | 최저가 |

5. 소싱 MD 액션 아이템
   - 긴급 발주 필요 (우리만 품절)
   - 가격 조정 기회 (경쟁몰만 품절)
   - 대체 상품 제안
   - 브랜드사 문의 필요 (양쪽 품절)
```

### 5단계: 지식베이스 업데이트
- competitor-tracking.md의 가격 정보 업데이트
- 품절 이력 기록 (장기 품절 추적용)

## 알림 기준

| 등급 | 조건 | 대응 |
|------|------|------|
| 긴급 | 베스트셀러 품절, 시즌 핵심상품 품절, 우리만 품절 | 즉시 보고 |
| 주의 | 카테고리 품절률 30% 초과, 브랜드 전량 품절 | 당일 대응 |
| 참고 | 경쟁몰 신상품 입고, 장기 품절(2주+) | 주간 리포트 |

## 리포트 디자인 규칙
- 짱베이스볼 디자인 시스템 적용 (design-system.md)
- 품절 = `var(--danger)` #C62828
- 재고 있음 = `var(--success)` #2E7D32
- 부분 품절 = `var(--warning)` #E65100
- 표 기반 레이아웃, 한눈에 파악 가능한 구성

## 출력 형식
- HTML 리포트 (필수) — `~/.claude/output/품절체크/` 폴더
- 요약 텍스트 (화면 출력) — 긴급 사항 + 액션 아이템
- 지식베이스 업데이트 (자동) — 새로 확인된 정보 반영
