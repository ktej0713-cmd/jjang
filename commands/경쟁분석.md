# 경쟁사 분석

짱베이스볼 기준으로 특정 상품/카테고리의 경쟁 환경을 분석하는 커맨드입니다.

## 입력
상품명, 카테고리, 또는 브랜드를 지정합니다. 예시:
- `/경쟁분석 미즈노 프로 BSS`
- `/경쟁분석 글러브 카테고리`
- `/경쟁분석 유소년 장비 시장`

인자: $ARGUMENTS

## 실행 절차

### 1단계: 내부 데이터 확인
- `~/.claude/knowledge/competitor-tracking.md`에서 기존 경쟁사 데이터 로드
- `~/.claude/knowledge/brand-catalog.md`에서 짱베이스볼 취급 현황 확인
- `~/.claude/knowledge/price-range-guide.md`에서 가격 포지션 확인

### 2단계: 외부 가격 조사
웹 검색으로 실시간 경쟁 가격 수집:
- 네이버 쇼핑 최저가 검색
- 야구야(yaguya.co.kr) 가격 확인
- 번개장터/중고나라 시세 참고 (고가 상품)

### 3단계: 분석 보고서 작성

#### 가격 비교표
| 채널 | 가격 | 배송 | 부가혜택 | 비고 |
|------|------|------|---------|------|

#### 경쟁 포지션 분석
- 짱베이스볼 가격 경쟁력 평가 (상/중/하)
- 경쟁사 대비 차별화 포인트
- 가격 조정 필요 여부 (10% 이상 차이 시 검토)

#### 시장 트렌드
- 해당 상품/카테고리 수요 동향
- 커뮤니티 반응 (엠엘비파크, 디시야갤 키워드)
- 시즌성 영향

### 4단계: 액션 아이템 제시
- 가격 대응 방안 (가격 조정 / 쿠폰 / 세트 구성)
- 콘텐츠 차별화 방안 (상세페이지 / 리뷰 / 가이드)
- 마케팅 키워드 추천

### 5단계: 지식베이스 업데이트
- 새로 확인된 가격 정보를 competitor-tracking.md에 반영
- 변동 이력 기록

## 가격 대응 원칙
1. 최저가 경쟁 지양 - 전문 콘텐츠 + 큐레이션 가치로 승부
2. 경쟁사 대비 10% 이상 높으면 가격 조정 검토
3. 단독 취급/오더 상품은 가격 경쟁 대상 아님
4. 쿠폰/적립금으로 실질 할인 제공 (표시가 유지)

## 출력 형식
- 가격 비교표 (필수)
- 경쟁 포지션 요약 (1-2문장)
- 액션 아이템 리스트 (실행 가능한 것만)
- MD 판단 근거
