# 품질체크

짱베이스볼 자사몰 상품 페이지 품질을 검수하고, 경쟁 야구몰과 비교하여 리포트를 생성합니다.

## 입력
상품 URL, 카테고리, 또는 "전체"를 지정합니다. 예시:
- `/품질체크` — 최근 등록/수정된 상품 페이지 체크
- `/품질체크 글러브` — 글러브 카테고리 상세페이지 전체
- `/품질체크 https://jjangbaseball.com/goods/view?no=123` — 특정 상품 페이지
- `/품질체크 경쟁비교 미즈노 프로 BSS` — 동일 상품 경쟁몰 비교

인자: $ARGUMENTS

## 실행 절차

### 0단계: 체크 대상 선별 (토큰 절약)

인자가 없으면(`/품질체크`) **새로 등록/수정된 상품만** 자동 선별합니다:

1. `~/.claude/output/상세페이지/` 폴더에서 최근 24시간 내 생성/수정된 HTML 파일 탐색
   ```bash
   find ~/.claude/output/상세페이지/ -name "*.html" -mtime -1 -type f
   ```
2. `~/.claude/output/품질체크/체크이력.log` 파일과 대조 → 이미 체크한 파일 제외
3. 신규 파일만 체크 대상으로 선정 (0건이면 "체크할 신규 상품 없음" 출력 후 종료)

체크 완료 후 이력 기록:
```
{파일명}|{체크일시}|{점수}|{등급}
```

인자가 있으면 해당 상품/카테고리를 직접 체크합니다 (이력 무관).

### 1단계: 내부 기준 로드
- `~/.claude/rules/copywriting.md` — 상세페이지 표준 구조 (10단계)
- `~/.claude/rules/brand-voice.md` — 브랜드 보이스 규칙
- `~/.claude/rules/design-system.md` — 디자인 시스템 표준
- `~/.claude/rules/design-rules.md` — 디자인 금지 사항
- `~/.claude/rules/seo-standards.md` — SEO 표준
- `~/.claude/knowledge/competitor-tracking.md` — 경쟁몰 정보

### 2단계: 자사몰 상품 페이지 품질 검수

WebFetch로 상품 상세페이지를 가져와서 4개 영역 100점 만점으로 평가:

#### 콘텐츠 품질 (40점)
| 항목 | 배점 | 체크 내용 |
|------|------|----------|
| 히어로 + 1줄 정의 | 5점 | 상품 특성이 즉시 파악되는 카피 |
| 필요 상황 | 5점 | 타겟별 구체적 사용 상황 |
| 스펙 표 | 5점 | 핵심 스펙 누락 없이 표 정리 |
| 체감 설명 | 5점 | 실사용자 관점, 스펙 나열 아님 |
| 포지션별 추천 | 5점 | 야구인이 판단 가능한 추천 |
| 사이즈 가이드 | 5점 | 실용적 선택 가이드 |
| 비교표 | 5점 | 같은 카테고리 다른 상품과 비교 |
| FAQ | 3점 | 실제 구매자 관점 질문 |
| 추천 포인트 | 2점 | 구매 결정 도움 여부 |

#### SEO 품질 (25점)
| 항목 | 배점 | 체크 내용 |
|------|------|----------|
| 메타 title | 3점 | [상품명] - [카테고리] 짱베이스볼 형식 |
| 메타 description | 3점 | 체감 중심 120~155자 |
| H1 태그 | 2점 | 1개, 상품명 포함 |
| JSON-LD Product | 5점 | name, brand, offers, description |
| JSON-LD FAQPage | 3점 | FAQ와 일치 |
| 이미지 alt | 3점 | 상품명 포함 alt |
| 내부 링크 | 3점 | 관련 상품/카테고리 |
| 네이버 쇼핑 최적화 | 3점 | 키워드, 카테고리 매핑 |

#### 디자인/UI 품질 (20점)
| 항목 | 배점 | 체크 내용 |
|------|------|----------|
| 모바일 반응형 | 5점 | 768px 이하 정상 표시 |
| 고도몰 에디터 호환 | 5점 | table 레이아웃, 시스템 폰트 |
| 가독성 | 5점 | 폰트/줄간격/색상 대비 |
| 디자인 시스템 준수 | 5점 | 브랜드 컬러/간격/컴포넌트 |

#### 야구 전문성 (15점)
| 항목 | 배점 | 체크 내용 |
|------|------|----------|
| 용어 정확성 | 5점 | 커뮤니티 실제 표현 |
| 타겟 적합성 | 5점 | 사회인/유소년/엘리트 구분 |
| 과장 없는 설명 | 5점 | 사실 기반, 과장 표현 없음 |

등급 기준:
| 등급 | 점수 | 판정 |
|------|------|------|
| S | 90~100 | 바로 라이브 가능 |
| A | 75~89 | 소폭 보완 후 라이브 |
| B | 60~74 | 주요 항목 보완 필요 |
| C | 40~59 | 재작성 검토 |
| D | 0~39 | 전면 재작성 |

### 3단계: 경쟁몰 비교 (인자에 "경쟁비교" 포함 시)

동일/유사 상품의 상세페이지를 경쟁몰에서 WebFetch로 가져와 비교:

| 사이트 | URL | 비교 방법 |
|--------|-----|----------|
| 야구야 | yaguya.co.kr | 상품 검색 → 상세페이지 품질 비교 |
| 야구넷 | yagooonet.com | 상품 검색 → 정보량 비교 |
| 베이스볼파크 | ballpark.co.kr | 상품 검색 → 구성 비교 |
| 네이버쇼핑 | search.shopping.naver.com | 상품 정보 표준 비교 |

비교 항목:
- 정보량 (스펙, 설명, 이미지 수)
- 사이즈/착용 가이드 유무
- FAQ / 비교표 유무
- 모바일 최적화 수준
- 구매 후기 활용도

### 4단계: 리포트 생성

출력 파일: `~/.claude/output/품질체크/품질리포트_{YYYYMMDD}.html`

리포트 구조:
```
1. 요약 대시보드
   - 체크 상품 수, 평균 점수, 등급 분포
   - S/A등급 비율 (목표: 80% 이상)

2. 상품별 점수표
   - 상품명 / 카테고리 / 총점 / 등급 / 주요 감점 사유

3. 항목별 분석
   - 가장 약한 영역 식별
   - 공통적으로 부족한 항목

4. 경쟁몰 비교 (실행 시)
   - 짱베이스볼 vs 경쟁몰 항목별 비교
   - 부족한 점 / 우위인 점

5. 콘텐츠 MD 액션 아이템
   - C등급 이하: 즉시 재작성
   - B등급: 특정 항목 보완
   - 경쟁몰 대비 개선 포인트
```

### 5단계: 로컬 HTML 파일 체크 (출력물 검수)

`~/.claude/output/상세페이지/` 폴더의 HTML 파일도 검수 대상:
- 최근 생성된 상세페이지 HTML 파일을 읽어서 품질 점수 산출
- 라이브 전 사전 검수 용도

## 출력 형식
- HTML 리포트 (필수) — `~/.claude/output/품질체크/` 폴더
- 요약 텍스트 (화면 출력) — 등급별 분포 + 액션 아이템
- 개선 제안 (상세) — 감점 항목별 구체적 수정 방향
