## 텍스트 안전 영역

- **PC 배너**: 좌우 20% 여백 → 핵심 콘텐츠는 **중앙 60%**에 배치
- **모바일 크롭** 시 좌우가 잘리므로 핵심 정보는 반드시 **중앙 집중**
- **텍스트 최소 크기**: PC 24px, 모바일 16px (가독성 확보)
- 메인 배너 기준 안전 영역: PC에서 좌우 384px씩 여백, 실질 콘텐츠 영역 1152px
- 모바일에서 좌우 144px씩 여백, 실질 콘텐츠 영역 432px

## 배너 텍스트 구조

1. **메인 카피** (1줄, 핵심 메시지)
2. **서브 카피** (1~2줄, 부연 설명)

## 배너 디자인 원칙

### 색상
- 디자인 시스템 팔레트 준수 (`design-system.md` 참조)
- **Primary**: `#1B2A4A` (헤더, 주요 배경)
- **Accent**: `#D4A843` (강조선, 포인트 컬러)
- **NEW**: `#E53935` (신상품, 세일 강조)

### 배경
- 그래디언트: `linear-gradient(135deg, var(--primary) 0%, var(--primary-light) 100%)`
- 또는 상품 이미지 활용

### 텍스트 가독성
- 이미지 위 텍스트 시 **반투명 오버레이 필수** (`rgba(27, 42, 74, 0.6)` 이상)
- 밝은 배경에는 `var(--gray-900)` 텍스트
- 어두운 배경에는 `var(--white)` 텍스트
- 텍스트 그림자 권장: `text-shadow: 0 2px 4px rgba(0,0,0,0.3)`

### 금지 사항
- **과장 광고 표현 금지**: "최저가 보장", "파격 할인", "업계 최초" 등
- 디자인 시스템 외 하드코딩 색상 사용 금지
- 텍스트만으로 가득 채운 배너 금지 (여백 30% 이상 확보)
- 3개 이상 폰트 혼용 금지

### 배너 교체 주기
- 시즌별 교체: `seasonal-calendar.md` 참조
- 메인 배너: 최소 2주 1회 교체 (신선도 유지)
- 기획전 배너: 기획전 기간에 맞춰 등록/해제
- 띠 배너: 이벤트 기간 한정 노출

## 고도몰 배너 등록 가이드

### 메인 배너 (슬라이드)
- **경로**: 관리자 > 디자인 > 배너관리 > 메인 슬라이드
- 권장 슬라이드 수: 3~5개 (너무 많으면 뒤 배너 노출률 급감)
- 자동 전환 시간: 4~5초
- PC/모바일 이미지 각각 등록

### 카테고리 배너
- **경로**: 각 카테고리 설정 > 상단 이미지
- 카테고리별 고유 비주얼 권장 (글러브=글러브 이미지, 배트=배트 이미지)
- alt 텍스트에 카테고리 키워드 포함 (SEO)

### 팝업 배너
- **경로**: 관리자 > 디자인 > 팝업관리
- 최대 동시 팝업: 2개 이하 (UX 저해 방지)
- "오늘 하루 보지 않기" 쿠키 설정 필수
- 모바일에서는 화면 50% 이하 크기 권장

### 띠 배너
- 상단 고정 또는 GNB 하단 배치
- 닫기(X) 버튼 필수
- 간결한 1줄 메시지 + CTA 링크

## 배너 체크리스트

배너 등록 전 아래 항목을 반드시 확인합니다:

- [ ] PC/모바일 사이즈 규격 준수 여부
- [ ] 용량 제한 이내 여부
- [ ] 텍스트 안전 영역 내 핵심 정보 배치 여부
- [ ] 디자인 시스템 색상 준수 여부
- [ ] 과장 광고 표현 포함 여부 (금지)
- [ ] CTA 버튼/링크 정상 동작 여부
- [ ] alt 텍스트 작성 여부 (SEO)
- [ ] 모바일에서 텍스트 가독성 확인
- [ ] 배너 링크 URL 정확성 확인
- [ ] 기간 한정 배너의 경우 종료일 설정 확인
