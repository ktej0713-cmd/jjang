# 네이버 쇼핑 카테고리 매핑

> 짱베이스볼(jjangbaseball.com) 카테고리와 네이버 쇼핑 EP 카테고리를 매핑하는 문서입니다.
> 에이전트가 EP 피드 생성, 상품명 최적화, 카테고리 매칭 작업 시 참조합니다.

## EP 피드 기본 정보

| 항목 | 내용 |
|------|------|
| EP 버전 | 3.0 |
| 수집 주기 | 일 4회 (01시, 10시, 14시, 18시) |
| 카테고리 코드 형식 | 8자리 숫자 (예: 50000803) |
| 코드 체계 | 네이버 쇼핑 파트너센터 기준 leafCategoryId |
| 최종 확인 방법 | 네이버 커머스 API > 전체 카테고리 조회 API |
| 파트너센터 URL | https://adcenter.shopping.naver.com |
| 커머스 API 센터 | https://apicenter.commerce.naver.com |

### 카테고리 코드 확인 방법 (3가지)

1. **커머스 API 호출**: `GET /v1/categories` 엔드포인트로 전체 카테고리 JSON 조회 후 "야구" 필터링
2. **파트너센터 엑셀 다운로드**: 파트너센터 > 상품관리 > 상품정보 수신현황 > [네이버쇼핑 카테고리ID] 버튼 > 엑셀 다운로드
3. **쇼핑 URL 직접 확인**: 네이버 쇼핑에서 해당 카테고리 진입 후 URL의 `catId=` 파라미터 값 확인

## 매핑표

> **중요**: 아래 카테고리 코드는 네이버 쇼핑 파트너센터/커머스 API에서 **반드시 직접 확인 후 갱신**해야 합니다.
> 카테고리 코드 열에 `[확인필요]`로 표시된 항목은 파트너센터에서 최종 검증이 필요합니다.
> 네이버 쇼핑은 카테고리를 수시로 개편하므로, 분기별 점검을 권장합니다.

## 매핑 시 주의사항

### 리프 카테고리 원칙
- 네이버 쇼핑 EP에서는 **반드시 리프(최하위) 카테고리 코드**를 입력해야 합니다.
- 예: "스포츠/레저 > 야구"(중분류)가 아닌 "스포츠/레저 > 야구 > 야구글러브 > 투수/올라운드용"(리프)
- 리프가 아닌 코드를 입력하면 EP 수집이 거부됩니다.

### 카테고리 미존재 시 대응
- 짱베이스볼의 세분류가 네이버 쇼핑에 없는 경우, **한 단계 상위 카테고리**로 매핑합니다.
- 예: "심판화"가 없으면 "야구화" 상위로 매핑
- 예: "주루장갑"이 없으면 "야구장갑" 또는 "배팅장갑"으로 매핑

### 1:N 매핑 허용
- 짱베이스볼 1개 카테고리가 네이버 쇼핑 여러 카테고리에 매핑될 수 있습니다.
- 예: "의류 > 바람막이"와 "의류 > 점퍼"는 네이버에서 같은 코드일 수 있음

### 유소년/커스텀 처리
- 네이버 쇼핑에 별도 유소년/커스텀 카테고리가 없으므로, 일반 카테고리 코드를 사용합니다.
- **상품명**과 **상품 속성**에서 "유소년", "커스텀", "오더" 등을 명시하여 검색 구분합니다.

## 상품명 SEO 패턴 가이드

> 네이버 쇼핑에서 상품명은 검색 노출의 핵심입니다.
> 아래 패턴을 카테고리별로 적용하면 네이버 쇼핑 검색 최적화에 유리합니다.

### 상품명 기본 규칙

| 규칙 | 설명 |
|------|------|
| 글자 수 | 50~100자 권장 (최대 100자) |
| 구조 | [브랜드] [모델명] [핵심스펙] [용도/대상] [카테고리키워드] |
| 금지 | 특수문자 남용, "최저가/할인/무료배송" 등 프로모션 문구 |
| 필수 | 브랜드명 + 모델명 반드시 포함 |

### 카테고리별 상품명 패턴

#### 글러브
```
[브랜드] [모델명] [포지션] [크기] [소재/웹] 야구글러브
```
**예시**: 미즈노 프로 셀렉트 GPS1-600S 투수용 12인치 T-NET웹 야구글러브

#### 배트
```
[브랜드] [모델명] [소재] [길이/무게] [밸런스] 야구배트
```
**예시**: 이스턴 ADV 360 카본 BBCOR 33/30 미들밸런스 야구배트

#### 야구화
```
[브랜드] [모델명] [종류] [사이즈범위] 야구화
```
**예시**: 미즈노 프랜차이즈 9 스파이크 금속징 260-290 야구화

#### 보호장비
```
[브랜드] [모델명] [부위] [사이즈] [대상] 야구보호장비
```
**예시**: 에보실드 프로-SRZ 배팅 암가드 L 성인용 야구보호장비

#### 배팅장갑
```
[브랜드] [모델명] [타입] [사이즈] 배팅장갑
```
**예시**: 미즈노 프로 셀렉션 양손 M 배팅장갑

#### 야구공
```
[브랜드] [모델명] [용도] [타수] 야구공
```
**예시**: 미즈노 공인 경식구 시합용 1타(12개입) 야구공

#### 포수장비
```
[브랜드] [모델명] [종류] [사이즈/대상] 포수장비
```
**예시**: 미즈노 사무라이 포수장비 세트 성인용 포수보호장비

#### 유소년 상품
```
[브랜드] [모델명] 유소년 [종류] [사이즈] [카테고리키워드]
```
**예시**: 미즈노 프로스펙트 유소년 야구글러브 10인치 내야수용 주니어글러브

#### 커스텀 오더
```
[브랜드] 오더 [종류] [핵심스펙] 커스텀 [카테고리키워드]
```
**예시**: 미즈노 오더글러브 투수용 12인치 커스텀 맞춤 야구글러브

## 카테고리 코드 갱신 절차

### 최초 설정
1. 네이버 커머스 API 또는 파트너센터에서 전체 카테고리 엑셀 다운로드
2. "스포츠/레저 > 야구" 하위 카테고리 필터링
3. 본 매핑표의 `[확인필요]` 항목을 실제 코드로 대체
4. 고도몰5 관리자 > 네이버 쇼핑 EP 설정에서 카테고리 코드 입력

### 정기 점검 (분기별)
1. 네이버 쇼핑 파트너센터에서 카테고리 변경 공지 확인
2. 커머스 API로 현재 카테고리 트리 재조회
3. 변경된 코드가 있으면 본 문서 및 고도몰 EP 설정 동시 업데이트
4. 업데이트 이력에 변경 내용 기록

### 고도몰5 EP 설정 경로
```
고도몰5 관리자 > 마케팅 > 네이버쇼핑 > 상품DB URL(EP) 관리
> 카테고리 매칭 > 네이버 카테고리 ID 입력
```
