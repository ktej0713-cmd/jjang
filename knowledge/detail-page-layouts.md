## 1. 배트

현행 레이아웃을 표준으로 유지합니다. 다른 카테고리의 기준점입니다.

### 섹션 순서

| 순서 | SEC 마커 | 클래스 / 구조 | 내용 |
|------|---------|--------------|------|
| 1 | `SEC01` | `.jd-hero` + `.jd-hero-text` | 히어로 이미지 + 제품 정의 |
| 2 | `SEC02` | `.jd-situation-list` | 이런 분께 맞습니다 (필요상황) |
| 3 | `SEC03` | `.jd-table` | 핵심 스펙표 |
| 4 | `SEC04` | `.jd-material-box` + `.jd-factor-grid` | 소재/구조 심화 |
| 5 | `SEC05` | `.jd-feel-list` | 체감설명 |
| 6 | `SEC06` | `.jd-position-grid` + `.jd-position-card` | 타자유형별 추천 (3열 카드) |
| 7 | `SEC07` | `.jd-compare-table` (사이즈용) | 사이즈 선택 가이드 |
| 8 | `SEC08` | `.jd-compare-table` (비교용) | 모델 vs 모델 / vs 일반 비교 |
| 9 | `SEC09` | `details.jd-faq-item` | FAQ |
| 10 | `SEC10` | `.jd-recommend-list` | 추천포인트 |

### 배트 전용 클래스

| 클래스명 | 용도 |
|---------|------|
| `.jd-material-box` | 소재 정의 박스 (소재명 + 원소 설명 등 강조 블록) |
| `.jd-material-title` | 소재 박스 내 타이틀 |
| `.jd-factor-grid` | 소재 특성 카드 3열 그리드 |
| `.jd-factor-card` | 소재 특성 카드 개별 아이템 |
| `.jd-factor-name` | 특성 카드 키워드 (영문 대문자 레이블) |
| `.jd-factor-desc` | 특성 카드 설명 |
| `.jd-position-grid` | 타자유형별 카드 3열 그리드 |
| `.jd-position-card` | 타자유형 카드 개별 아이템 |
| `.jd-pos-name` | 카드 내 유형명 |
| `.jd-pos-rating` | 카드 내 적합도 표시 |
| `.jd-pos-desc` | 카드 내 설명 |
| `.jd-compare-table` | 비교표 (사이즈 가이드 / 모델 비교 공용) |

### 작성 주의사항

- `SEC04` 소재 심화: 스칸듐 계열은 원소 정보 + 구간별 특성 블록을 사용합니다. 목재 배트는 목재 종류별 특성 비교로 대체합니다.
- `SEC06` 타자유형 카드: 포지션이 아닌 "타격 유형"(컨택형/파워형/스피드형 등) 기준으로 작성합니다.
- `SEC07` 사이즈 가이드: 인치/온스 기준 표로 구성합니다. KBSA 공인 여부를 별도 행으로 포함합니다.
- `SEC08` 비교표: 타 모델이 없을 경우 "일반 알로이 vs 이 배트" 구도로 작성합니다.

## 2. 의류

### 섹션 순서

| 순서 | SEC 마커 | 클래스 / 구조 | 내용 |
|------|---------|--------------|------|
| 1 | `SEC01` | `.jd-hero` + `.jd-hero-text` | 착용 이미지 중심 히어로 + 제품 정의 |
| 2 | `SEC02` | `.jd-situation-list` | 필요상황 |
| 3 | `SEC03` | `.jd-table` | 핵심 스펙표 |
| 4 | `SEC04` | `.jd-feel-list` | 체감설명 (착용감 중심) |
| 5 | `SEC05` | `.jd-wear-grid` + `.jd-wear-card` | 착용 상황 가이드 |
| 6 | `SEC06` | `.jd-size-table` + `.jd-size-note` | 사이즈 가이드 (신체 치수 상세) |
| 7 | `SEC07` | `.jd-material-compare` | 소재 비교 (이 소재 vs 일반 소재) |
| 8 | `SEC08` | `details.jd-faq-item` | FAQ |
| 9 | `SEC09` | `.jd-recommend-list` | 추천포인트 |

### 배트 대비 변경사항

| 배트 섹션 | 의류 섹션 | 변경 이유 |
|----------|---------|---------|
| `SEC04` 소재/구조 심화 (`jd-material-box`) | `SEC07` 소재 비교 (`jd-material-compare`) | 의류는 원소 설명 불필요. "기능성 소재 vs 코튼" 비교가 구매 결정에 더 직접적입니다. |
| `SEC06` 타자유형별 카드 (`jd-position-grid`) | `SEC05` 착용 상황 가이드 (`jd-wear-grid`) | 포지션보다 "경기 중/훈련/이동" 등 착용 상황이 더 유효합니다. |
| `SEC08` 모델 비교표 (`jd-compare-table`) | 제거 | 의류 모델 간 비교는 사이즈 가이드로 대체합니다. |

### 의류 전용 클래스

| 클래스명 | 용도 |
|---------|------|
| `.jd-wear-grid` | 착용 상황 카드 3열 그리드 |
| `.jd-wear-card` | 착용 상황 카드 개별 아이템 |
| `.jd-wear-situation` | 카드 내 상황명 (예: 경기 중, 훈련, 이동) |
| `.jd-wear-desc` | 카드 내 한 줄 설명 |
| `.jd-size-table` | 신체 치수 기반 사이즈 표 |
| `.jd-size-note` | 사이즈 선택 주의사항 박스 |
| `.jd-material-compare` | 소재 비교 블록 (2열 구조) |

### 작성 주의사항

- `SEC01` 히어로: 배트는 제품 단독 컷이 주가 되지만, 의류는 착용 이미지(동작 중 컷)가 우선입니다. 이미지가 없을 경우 제품 컷 + 착용 상황 텍스트로 대체합니다.
- `SEC05` 착용 상황 카드: 포지션 카드와 달리 적합도(`jd-pos-rating`) 행을 제거합니다. 카드는 배경색(밝은 계열 교차)으로 상황을 구분합니다.
- `SEC06` 사이즈 가이드: 키/몸무게/가슴둘레/허리둘레를 기준으로 작성합니다. 브랜드별 핏 특성(슬림/레귤러/루즈) 한 줄을 표 상단에 명시합니다.
- `SEC07` 소재 비교: 이 제품 소재(기능성)와 일반 소재(코튼 등)를 2열로 비교합니다. 각 항목은 흡수성/건조속도/신축성/내구성/관리 5개 행으로 구성합니다.

### `SEC06` 포지션 카드 내용 구성

배트의 타자유형 카드와 같은 `.jd-position-grid` / `.jd-position-card` 구조를 사용하되, 카드 내용을 아래로 구성합니다.

| 요소 | 클래스 | 내용 예시 |
|------|-------|---------|
| 포지션명 | `.jd-pos-name` | 투수, 내야수, 외야수, 포수 |
| 추천 웹사이즈 | `.jd-pos-rating` | 11.5인치 / 웹: H-웹 |
| 설명 | `.jd-pos-desc` | 깊은 포켓, 빠른 볼 처리에 적합 |

### 작성 주의사항

- `SEC04` 가죽 심화: 스티어하이드/킵스킨/합성가죽의 가격대별 포지셔닝을 포함합니다. 탄닝 방식(식물성/크롬)을 한 줄로 설명합니다.
- `SEC07` 길들이기 가이드: 번인 오일 → 포켓 잡기 → 보관 순서로 3~5단계로 구성합니다. 금지 방법(전자레인지 등)을 주의사항 박스(`.jd-note`)로 표기합니다.
- `SEC08` 비교표: 같은 가격대 타사 글러브와 소재/무게/포켓 깊이 기준으로 비교합니다. 타사명을 직접 명시할 경우 객관적 스펙 비교만 허용합니다.

### `SEC06` 포지션 카드 내용 구성

보호 필요도를 상황별로 제시합니다. `.jd-position-grid` / `.jd-position-card` 구조 유지.

| 요소 | 클래스 | 내용 예시 |
|------|-------|---------|
| 상황명 | `.jd-pos-name` | 타자, 주자, 포수, 내야 수비 |
| 필요도 | `.jd-pos-rating` | 필수 / 권장 / 선택 |
| 설명 | `.jd-pos-desc` | 파울팁 직격 빈도가 높아 필수 착용입니다. |

### 작성 주의사항

- `SEC04` 보호 구조: 부위별(예: 헬멧이면 쉘/폼/턱받침, 슬라이딩장갑이면 손바닥/손목) 로 구분합니다. 각 부위의 소재명과 보호 원리를 한 줄로 설명합니다.
- `SEC07` 사이즈 가이드: 헬멧은 머리둘레(cm), 신체보호대는 가슴둘레/키, 정강이보호대는 다리 길이 기준으로 작성합니다. 장비 유형에 따라 표 기준이 달라지므로 적합한 신체 부위 기준을 명확히 표시합니다.

### `SEC06` 그라운드별 카드 내용 구성

`.jd-position-grid` / `.jd-position-card` 구조를 재사용합니다.

| 요소 | 클래스 | 내용 예시 |
|------|-------|---------|
| 그라운드명 | `.jd-pos-name` | 흙 그라운드, 인조잔디, 실내 훈련장 |
| 적합도 | `.jd-pos-rating` | 최적 / 사용 가능 / 부적합 |
| 설명 | `.jd-pos-desc` | 금속 스터드로 접지력이 높습니다. 인조잔디에서는 바닥 손상 우려가 있습니다. |

### 작성 주의사항

- `SEC04` 밑창 구조: 아웃솔 소재(TPU/고무), 스터드 형태(금속/비금속/몰드), 스터드 수와 배치를 설명합니다. 리그 규정상 금속 스터드 제한 여부를 주의사항 박스(`.jd-note`)로 포함합니다.
- `SEC07` 사이즈 가이드: 발길이(mm)와 발볼 너비(A~4E)를 동시에 표기합니다. 브랜드별 라스트(목형) 특성을 한 줄로 명시합니다.
- `SEC08` 비교표: 동일 브랜드 상위/하위 모델과 가격/무게/스터드/소재 기준으로 비교합니다.

## 6. 장갑 (배팅/수비)

### 섹션 순서

| 순서 | SEC 마커 | 클래스 / 구조 | 내용 |
|------|---------|--------------|------|
| 1 | `SEC01` | `.jd-hero` + `.jd-hero-text` | 히어로 이미지 + 제품 정의 |
| 2 | `SEC02` | `.jd-situation-list` | 필요상황 |
| 3 | `SEC03` | `.jd-table` | 핵심 스펙표 |
| 4 | `SEC04` | `.jd-feel-list` | 체감설명 (그립감, 내구성) |
| 5 | `SEC05` | `.jd-glove-use-grid` + `.jd-glove-use-card` | 용도별 추천 (배팅/수비/겸용) |
| 6 | `SEC06` | `.jd-size-table` + `.jd-size-note` | 사이즈 가이드 (손 둘레 측정법 포함) |
| 7 | `SEC07` | `details.jd-faq-item` | FAQ |
| 8 | `SEC08` | `.jd-recommend-list` | 추천포인트 |

### 배트 대비 변경사항

| 배트 섹션 | 장갑 섹션 | 변경 이유 |
|----------|---------|---------|
| `SEC04` 소재/구조 심화 | 제거 | 장갑 소재 설명은 스펙표(SEC03)와 체감설명(SEC04)으로 충분합니다. |
| `SEC06` 타자유형 카드 | `SEC05` 용도별 추천 (`jd-glove-use-grid`) | 배팅/수비/겸용 구분이 장갑의 핵심 구매 기준입니다. |
| `SEC08` 비교표 | 제거 | 장갑은 사이즈 선택 정확도가 비교표보다 중요합니다. |

### 장갑 전용 클래스

| 클래스명 | 용도 |
|---------|------|
| `.jd-glove-use-grid` | 용도별 추천 카드 그리드 (3열) |
| `.jd-glove-use-card` | 용도 카드 개별 아이템 |
| `.jd-glove-use-name` | 카드 내 용도명 (배팅용, 수비용, 겸용) |
| `.jd-glove-use-desc` | 카드 내 설명 |
| `.jd-size-table` | 손 둘레 기준 사이즈 표 |
| `.jd-size-note` | 손 둘레 측정 방법 안내 박스 |

### `SEC05` 용도별 카드 내용 구성

| 요소 | 클래스 | 내용 예시 |
|------|-------|---------|
| 용도명 | `.jd-glove-use-name` | 배팅 전용, 수비 전용, 겸용 |
| 설명 | `.jd-glove-use-desc` | 배트 진동 흡수를 위한 패딩이 손바닥 전면에 있습니다. |

적합도 표시(`.jd-pos-rating`)는 이 카드에서 사용하지 않습니다. 배경색만으로 구분합니다.

### 작성 주의사항

- `SEC06` 사이즈 가이드: 손 둘레 측정법(새끼손가락 아래 관절 기준 줄자 감기)을 `.jd-size-note`에 포함합니다. 사이즈 표는 둘레(cm)와 S/M/L/XL 대응으로 구성합니다.
- `SEC05` 용도 카드: 배팅용은 그립 패딩 위치, 수비용은 손가락 노출 여부와 미끄럼 방지 처리, 겸용은 두 조건의 절충 특성을 설명합니다.
- 배팅장갑과 수비장갑이 동일 상세페이지에 함께 표시되는 경우, `SEC03` 스펙표를 용도별 탭 없이 각 용도 행을 구분선으로 나누어 작성합니다.
