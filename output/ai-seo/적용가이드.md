# AI 추천 노출 개선 - 고도몰 적용 가이드

## 생성된 파일 목록

| 파일 | 용도 | 적용 위치 |
|------|------|----------|
| `llms.txt` | AI에게 사이트 자기소개 | 서버 루트 디렉토리 |
| `organization-jsonld.html` | 조직/매장/검색 스키마 | layout/head.html |
| `product-jsonld.html` | 상품 구조화 데이터 | goods/goods_view.html |
| `meta-og-tags.html` | 메타태그 + OG 태그 | layout/head.html + goods/goods_view.html |

---

## 적용 순서

### 1단계: llms.txt 업로드 (가장 쉬움)

**방법**: FTP로 서버 루트에 파일 업로드

1. FTP 접속 (호스팅 관리자에서 FTP 정보 확인)
2. 루트 디렉토리 (`/public_html/` 또는 `/www/`)에 `llms.txt` 업로드
3. 확인: `https://www.jjangbaseball.com/llms.txt` 접속 시 내용이 보이면 성공

> llms.txt는 AI(ChatGPT, Claude, Perplexity)가 사이트를 방문할 때
> "이 사이트가 뭐하는 곳인지" 빠르게 파악하는 자기소개서입니다.

---

### 2단계: 조직 정보 JSON-LD 추가

**방법**: 고도몰 관리자 > 디자인 > 스킨 편집

1. 고도몰 관리자 로그인
2. **디자인 > 디자인 스킨 설정 > 스킨 편집**
3. 좌측 파일 목록에서 **layout/head.html** 선택
4. `</head>` 바로 위에 `organization-jsonld.html`의 내용을 복사-붙여넣기
5. 저장

**포함되는 스키마:**
- Organization (짱베이스볼 회사 정보)
- WebSite + SearchAction (사이트 검색 기능)
- SportingGoodsStore (오프라인 매장 + 영업시간)
- BreadcrumbList (빵부스러기 네비게이션)

---

### 3단계: 공통 메타 태그 추가

**방법**: 같은 layout/head.html에 추가

1. `layout/head.html` 열기
2. `<title>` 태그 바로 아래에 `meta-og-tags.html`의 **[파일 1] 부분만** 복사-붙여넣기
3. 저장

```html
<!-- 이 부분만 복사 -->
<meta name="description" content="짱베이스볼 - 야구를 아는 사람이 운영하는 ...">
<meta property="og:type" content="website">
<meta property="og:site_name" content="짱베이스볼">
<meta property="og:locale" content="ko_KR">
<meta name="twitter:card" content="summary_large_image">
```

---

### 4단계: 상품 상세 페이지 JSON-LD + 메타 태그

**방법**: goods/goods_view.html 수정

1. 스킨 편집에서 **goods/goods_view.html** 선택
2. 파일 최하단 (`</body>` 위 또는 `{ # footer }` 아래)에:
   - `product-jsonld.html` 전체 내용 복사-붙여넣기
   - `meta-og-tags.html`의 **[파일 2] 부분** 복사-붙여넣기
3. 저장

---

## 적용 후 확인 방법

### JSON-LD 검증
1. Google 리치 결과 테스트: https://search.google.com/test/rich-results
2. 상품 페이지 URL 입력 후 테스트 실행
3. "Product" 스키마가 감지되면 성공

### OG 태그 검증
1. https://developers.facebook.com/tools/debug/ 에서 URL 입력
2. og:title, og:image, og:description이 표시되면 성공

### llms.txt 확인
1. 브라우저에서 `https://www.jjangbaseball.com/llms.txt` 접속
2. 내용이 보이면 성공

## 주의사항

1. **스킨 백업 필수**: 수정 전 기존 파일 내용을 메모장에 복사해두세요
2. **치환코드 주의**: `{=goodsView['goodsNm']}` 같은 코드는 그대로 유지해야 합니다
3. **캐시 초기화**: 수정 후 고도몰 관리자 > 운영관리 > 캐시관리에서 캐시 삭제
4. **모바일 확인**: PC/모바일 스킨이 별도인 경우 모바일 스킨에도 동일 적용
