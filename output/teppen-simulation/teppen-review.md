## 1. PC 배너 — banner-teppen-order-simulation.html

### 판정: PASS (경미 수정 권고 1건)

**잘된 점**
- 배경 이미지(글러브)가 우측에 배치되고 좌측에 텍스트가 오는 구성은 실제 쇼핑몰 기획전 배너와 동일한 레이아웃이다. AI가 만든 티가 나지 않는다.
- "내 글러브, 직접 디자인해보세요"는 과장 없이 시뮬레이션 서비스를 정확히 설명한다. 야구인이 봤을 때 거부감 없다.
- 스펙 라인(11 모델 / 16 웹 / 18 레이스컬러 / 2 레더)이 한눈에 들어온다. 엘리트 선수가 봤을 때 "구체적 정보가 있네"하고 클릭할 만하다.
- "오사카 자사공장 장인이 만드는"이라는 표현은 사실에 기반하며, 과장이 아니다.

**야구인 한줄 평**: 사회인야구 카페에 올라와도 "광고 같다"는 반응보다 "테펜 어디서 사?" 반응이 먼저 나올 수준.

### 수정 포인트

#### 3-1. [FAIL 수준] 가격 표기에 "세별(稅別)" 일본식 표현 사용 — 한국 소비자에게 혼란

**위치**: 커스터마이징 옵션 섹션 > price-note 영역 (670~674행)

**현재**:
```
기본 가격 (세별, 稅別)
- 북미산 스티어 레더 사용 모델: 52,000엔 (세별)
- 더치비어 킵 레더 사용 모델: 60,000엔 (세별)
```

**문제**: "세별(稅別)"은 일본식 한자어로, 한국 야구인은 이 용어를 모른다. 엠엘비파크에 올리면 "세별이 뭔데?"라는 반응이 나온다. "세전(tax excluded)" 또는 "부가세 별도"가 한국어 표현이다.

**수정안**:
```
기본 가격 (일본 현지가, 부가세 별도)
- 북미산 스티어 레더 사용 모델: 52,000엔 (부가세 별도)
- 더치비어 킵 레더 사용 모델: 60,000엔 (부가세 별도)
```

**동일 문제 추가 위치** (모두 동일하게 수정):
- 혁끈 교체: "1본당 1,000엔 (세별)" → "1본당 1,000엔 (부가세 별도)"
- 자수: "1개소 2,000엔 (세별)" → "1개소 2,000엔 (부가세 별도)"
- 주의사항 섹션 혁끈 교체: "1본당 1,000엔 (세별)" → "1본당 1,000엔 (부가세 별도)"
- 주의사항 섹션 자수: "1개소 2,000엔 세별" → "1개소 2,000엔 부가세 별도"
- 주의사항 섹션 가격 기준: "모든 가격은 세별(稅別) 표기입니다" → "모든 가격은 일본 현지가 기준이며, 부가세가 별도입니다"

#### 3-2. [수정 필요] 레더(가죽) 명칭 불일치

**위치 1**: PC 배너 spec-line (170행) — "2 레더"
**위치 2**: 상세페이지 커스터마이징 옵션 price-note (671~672행) — "북미산 스티어 레더", "더치비어 킵 레더"
**위치 3**: 소싱 MD 가격 리서치 — "북미산 스테어 레더", "유러피언 킵 레더"

**문제**:
- "스티어" vs "스테어" — 원어는 "Steer"(거세한 수소). 야구 커뮤니티에서는 "스티어"가 더 보편적이다. 통일이 필요하다.
- "더치비어 킵 레더"라는 명칭이 생소하다. 테펜 공식 사이트에서 이 명칭을 사용하는지 확인이 필요하다. 일반적으로는 "유러피언 킵(Kip) 레더"라고 한다. "더치비어"가 정확한 고유 명칭이 아니라면 혼란을 줄 수 있다.

**수정안**:
- "스티어"로 통일 (커뮤니티 보편 표기)
- "더치비어 킵 레더"가 테펜 공식 명칭이 맞는지 확인 후, 맞다면 "(유러피언 킵)" 병기. 아니라면 "유러피언 킵 레더"로 교체.

#### 3-3. [수정 필요] 모델 특징 설명에 일본어 혼용 — 통일성 부족

**위치**: 오더 가능 모델 테이블 (617~659행)

**현재**:
- RM6: "当て捕り(당겨잡기)" — 일본어 원문 + 한국어 번역 병기
- ATM2: "쓰카미捕り(움켜잡기)" — 일본어 원문(한자+히라가나 혼합) + 한국어 번역

**문제**:
1. "当て捕り"를 "(당겨잡기)"라고 번역했는데, 이것은 정확하지 않다. "아테도리(当て捕り)"는 "글러브에 공을 갖다 대듯이 잡는 것"으로, "당겨잡기"보다는 "대어잡기" 또는 그냥 "아테도리"가 야구인들 사이에서 통용되는 표현이다.
2. "쓰카미捕り"는 일본어 히라가나/한자가 섞여 있어 어색하다. 유모미 가공 섹션에서는 "쓰카미도리(掴み捕り)"라고 한글+한자 괄호 표기를 했는데, 모델표에서는 표기 방식이 다르다.

**수정안**:
- RM6: "아테도리(当て捕り) 기본형. 얕은 포켓, 빠른 송구에 유리"
- ATM2: "쓰카미도리(掴み捕り) 기본형. 깊은 포켓, 강한 타구 대응"

#### 3-4. [수정 필요] "한 땀 한 땀" 표현 — 과장 느낌

**위치**: 히어로 섹션 hero-desc (517행)

**현재**: "오사카 자사 공장에서 한 땀 한 땀, 내 손에 맞는 글러브를 만듭니다"

**문제**: "한 땀 한 땀"은 수공예 브랜드가 지나치게 자주 쓰는 관용구다. 디시야갤에서 "한 땀 한 땀 ㅋㅋ"이라고 반응할 수 있다. 짱베이스볼 브랜드 보이스는 과장을 지양한다.

**수정안**: "오사카 미노시 자사 공장에서 직접 제작합니다"
- "미노시"를 넣으면 구체성이 올라가고, 과장 어감이 사라진다.

#### 3-5. [수정 필요] 본체 컬러 선택지가 3개뿐인데 시뮬레이션 가치가 있는지 의문

**위치**: 커스터마이징 옵션 테이블 > 본체 컬러 (698~699행)

**현재**: "딥오렌지 / 블랙 / 블론드"

**문제**: 본체 컬러가 3색뿐이라면, 시뮬레이션에서 "17가지 항목"을 강조한 것 대비 본체 컬러 선택의 폭이 좁아 보인다. 이것 자체가 문제는 아니지만, 실제 시뮬레이션에서 선택 가능한 본체 컬러가 정말 3개인지 확인이 필요하다. 만약 더 있다면 누락이고, 정말 3개라면 그대로 두되 웹이나 레이스에서 조합 폭이 넓다는 것을 보조 설명으로 추가하는 것이 좋다.

**수정안** (3색이 맞는 경우):
"딥오렌지 / 블랙 / 블론드" 아래에 다음 보조 설명 추가:
"※ 웹 16종, 레이스 18색, 스티치 17색 조합으로 다양한 디자인이 가능합니다"

#### 3-6. [수정 권고] 납기 표현이 섹션마다 미묘하게 다름 — 통일 필요

**위치 비교**:
- STEP 4 (814행): "제작 기간은 약 8~14주이며, 시기에 따라 달라질 수 있습니다"
- 주의사항 납기 (850행): "지금 주문 시 약 5~6월 수령 예정 (8~14주 소요)"
- 소싱 MD 가격 리서치: "주문 후 90~100일"

**문제**:
- 8주 = 56일, 14주 = 98일. 소싱 MD의 "90~100일"과 대략 일치하지만, 최소값(8주=56일)과 최대값(100일)의 차이가 크다.
- "지금 주문 시 약 5~6월 수령"은 현재 3월 19일 기준으로 계산하면 8주 후가 5월 14일, 14주 후가 6월 25일이므로 정직한 표현이다. 이 부분은 문제없다.

**수정안**: "8~14주"를 기본 표현으로 통일하고, STEP 4와 주의사항 모두 동일한 문구를 사용한다. "90~100일"은 리서치 문서의 표현이므로 상세페이지에는 "8~14주"가 적절하다. (주 단위가 고객 체감에 더 직관적)

#### 3-7. [수정 권고] 원화 가격 안내가 불충분 — 가격 민감 고객 이탈 가능성

**위치**: 커스터마이징 옵션 price-note (674행)

**현재**: "원화 판매가는 오더 접수 시 별도 안내드립니다 (환율 적용)"

**문제**: 사회인 야구 선수 입장에서 "대체 얼마인데?"가 첫 반응이다. 엔화 가격만 보고는 감이 안 온다. 소싱 MD 가격 리서치에 따르면 짱베이스볼 판매가는 780,000~820,000원 구간으로 포지셔닝 예정인데, 상세페이지에서 가격 범위조차 제시하지 않으면 채널톡 상담 전에 이탈할 수 있다.

**수정안**: 엔화 가격 아래에 참고 가격대를 추가한다.
```
※ 원화 기준 참고 가격: 약 78~82만원대 (모델/옵션에 따라 변동)
※ 정확한 금액은 오더 접수 시 안내드립니다 (환율 적용)
```

단, 이것은 소싱 MD / Lead MD와 가격 공개 범위에 대해 사전 협의가 필요한 사항이다. 가격이 확정되지 않았다면 현행대로 유지하되, 채널톡 응대 시 가격 문의에 즉시 답변할 수 있도록 준비해야 한다.
