## 5단계: 커스터마이징 프로세스 진행

**현재 상태**: 투수용(投手用) 글러브 선택 후 커스터마이징 옵션 표시

### 커스터마이징 옵션 발견

모달 다이얼로그를 통해 다음 옵션들이 표시되었습니다:

#### 1. **기본 모델 선택**
- 공식 제품명과 이미지로 표시
- 선택 가능한 모델 3가지 표시 (좌: 공통색상, 우: 특수색상)

#### 2. **본체 컬러 (グローブ色)**
- 빨강, 검정, 갈색 등 다양한 색상
- 체크박스 형식으로 중복 선택 가능하게 보임
- 최소 7가지 이상의 색상 옵션

#### 3. **웹/포켓 형태 (ウェブ)**
- 다양한 웹 형태의 패턴 표시
- 시각적으로 구분 가능한 아이콘으로 표현

#### 4. **스티치 색상 (ステッチ)**
- 주요색 (赤 - 빨강): 체크박스 형식
- 패턴 선택 가능 (여러 패턴 표시)

#### 5. **레이스 색상 (レース)**
- 빨강, 흰색 등 색상 선택
- 여러 패턴 옵션 제시

#### 6. **추가 옵션**
- 각 부위별 세부 조정 옵션 다수
- 스크롤로 더 많은 옵션 확인 가능
