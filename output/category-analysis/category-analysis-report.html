2 핵심 문제점 6가지

문제 1. 유소년 상품 이중 등록

유소년 글러브가 글러브(001) 중분류 "유소년"에도, 유소년용품(021) 중분류 "글러브"에도 존재합니다. 배트, 야구화, 장갑 등도 동일합니다.

영향: 검색 결과에서 같은 상품이 2번 노출되어 SEO 중복 페이지 이슈 발생. 학부모가 어디서 찾아야 할지 혼란. 재고 관리 시 어느 카테고리 기준인지 불명확.

문제 2. "공·트레이닝·케어·고글" 잡동사니 카테고리

야구공, 트레이닝용품, 케어/재활, 선글라스가 하나의 대분류에 묶여 있습니다. 성격이 완전히 다른 상품군이 혼합되어 있습니다.

영향: 네이버 쇼핑에서 "야구공"으로 검색하는 고객이 해당 카테고리를 찾기 어렵습니다. 경쟁몰(야중사, 야구매니아)은 야구공을 독립 대분류로 운영합니다.

문제 3. 의류 세분화 부족

"상의" 안에 스판언더셔츠, 티셔츠, 트레이닝상의, 방한의류가 모두 섞여 있습니다. 1,415개 상품이 7개 중분류에만 분산되어 있어 탐색이 어렵습니다.

영향: "야구 언더셔츠" 검색 시 직접 랜딩 불가. 트레이닝복과 경기복 구분이 안 되어 사용자가 직접 필터링해야 합니다.

문제 4. 심판용품 분산

심판장비가 보호장비(005)에, 심판의류가 의류(007)에, 심판화가 야구화(002)에 흩어져 있습니다.

영향: 심판이 장비를 한 번에 구매하려면 3개 카테고리를 돌아다녀야 합니다. 경쟁몰(야중사, 야구매니아)은 "포수/심판"으로 통합 운영합니다.

문제 5. 유소년 카테고리 네비게이션 후순위 배치

유소년용품이 대분류 목록 하단에 위치합니다. 경쟁몰(야용사, 야구매니아)은 유소년을 2번째에 배치합니다.

영향: 유소년 장비를 찾는 학부모의 첫 클릭 도달률 저하. 유소년은 객단가가 낮지만 반복 구매율이 높고 LTV가 높은 고객군입니다.

문제 6. 당일출고/빠른배송 카테고리 부재

경쟁몰(야중사 "당일출고", 야용사 "오늘출발")은 배송 기반 카테고리를 최상단에 배치하고 있으나 짱베이스볼에는 없습니다.

영향: 급하게 장비가 필요한 고객(대회 전, 장비 파손)의 전환 기회 손실.

6 유소년 상품 중복 해결 방안

방안: "유소년용품" 카테고리를 메인 카테고리로, 각 장비 카테고리에서 유소년 중분류 제거

Before: 유소년 글러브가 글러브(001) + 유소년(021) 양쪽에 등록
After: 유소년 글러브는 유소년용품(021) > 글러브에만 등록

실행 방법:
1. 글러브(001) 중분류에서 "유소년" 삭제
2. 배트(003) 중분류에서 "유소년배트" 삭제
3. 해당 상품들의 카테고리를 유소년용품(021)으로 일원화
4. 글러브(001) 페이지에 "유소년 글러브는 여기" 배너로 크로스링크

주의사항

  • 고도몰 관리자에서 상품별 카테고리 매핑을 변경해야 합니다 (대량작업 필요)
  • 기존 URL의 301 리다이렉트 설정이 필요합니다 (SEO 손실 방지)
  • 네이버 EP(상품 피드)에서 카테고리 코드 매핑도 함께 변경해야 합니다
  • 변경 후 2~3주간 검색엔진 인덱싱 모니터링이 필요합니다

  • 유소년 상품 중복 등록 일원화
  • 카테고리 변경 시 301 리다이렉트 설정

중기 (2~3개월)

  • 트레이닝·케어 독립 대분류 신설
  • 선글라스·악세사리 독립 대분류 신설
  • 당일출고 태그/필터 카테고리 검토

유소년 전진 배치 PASS 유소년 야구 인구 증가 추세. 학부모가 첫 방문 시 유소년 카테고리를 바로 찾을 수 있어야 합니다. 야용사·야구매니아가 이미 2번째에 배치한 건 검증된 패턴입니다.
포수·심판 통합 PASS 실제로 심판이 장비를 구매할 때 "심판" 검색 한 번으로 모든 장비를 찾고 싶어합니다. 현재처럼 3곳에 흩어져 있으면 이탈합니다.
야구공 독립 PASS 사회인 야구팀 운영진이 "야구공 1다스" 검색할 때 바로 찾을 수 있어야 합니다. 현재 "공·트레이닝·케어·고글"은 너무 길고 찾기 어렵습니다.
의류 세분화 PASS "스판 언더셔츠"와 "트레이닝 긴팔"은 용도가 완전히 다릅니다. 야구인이라면 구분을 기대합니다. 다만 중분류 수가 너무 많아지면 역효과이므로 5~7개가 적정입니다.
글러브 수리/중고 유지 PASS 글러브 수리는 짱베이스볼의 핵심 정체성입니다. "야구를 아는 몰"이라는 인식을 만드는 카테고리이므로 절대 제거하면 안 됩니다.
당일출고 카테고리 보류 실제 운영 가능 여부 확인이 필요합니다. 당일출고를 표방하면서 3~4일 걸리면 신뢰 손실이 큽니다. 물류 역량이 뒷받침된 후 도입을 권장합니다.

Lead MD 최종 판단

현재 짱베이스볼의 카테고리 구조는 "글러브 수리", "중고 글러브", "커스텀 오더 전 품목"이라는 강력한 차별 포인트를 갖고 있습니다. 이건 경쟁몰 어디에도 없는 강점이므로 반드시 유지해야 합니다.

반면, 유소년 배치 순서, 야구공·트레이닝 통합 문제, 의류 세분화 부족, 심판용품 분산은 개선이 필요합니다.

  • 유소년 카테고리를 네비게이션 2번째로 전진 배치
  • 포수장비 + 심판용품을 "포수·심판"으로 통합
  • "공·트레이닝·케어·고글"을 해체하고 야구공 독립
  • 의류 중분류 세분화 (언더셔츠, 트레이닝복 분리)
  • 유소년 상품 이중 등록 일원화
  • 글러브 수리·중고·커스텀 오더는 핵심 정체성으로 강화
