# 짱베이스볼 기획전/시즌 카테고리 구조 설계

## Context
짱베이스볼 쇼핑몰의 기획전 및 시즌별 카테고리 구조를 설계한다.
타겟은 사회인/유소년/엘리트 전 타겟 균등 비중.
최종 결과물은 `output/category-structure/` 폴더에 HTML로 생성.

## 카테고리 구조 방향

### 1. 시즌 기획전 (연간 캘린더 기반)
- **동계 시즌** (12~2월): 동계훈련 용품, 방한용품, 실내훈련 장비
- **개막 시즌** (3~4월): 신학기/개막 세트, 입문자 패키지, 팀 단체구매
- **하계 시즌** (5~8월): 여름 언더셔츠, 냉감용품, 수분보충
- **대회 시즌** (9~11월): 대회용 장비, 가을야구 기획전
- **입시 시즌** (8~10월): 체대입시/특기생 장비 세트

### 2. 타겟별 기획전
- **사회인 야구**: 주말리거 추천템, 포지션별 추천
- **유소년 야구**: 성장기 맞춤, 학부모 가이드
- **엘리트 선수**: 고급 장비, 프로스펙 라인

### 3. 상황/목적별 기획전
- **입문자 가이드**: 처음 시작하는 야구, 필수템 모음
- **포지션별 추천**: 투수/포수/내야/외야별 장비
- **팀 단체구매**: 유니폼, 팀 장비 세트
- **선물 기획전**: 야구인 선물 추천

## 산출물
- `output/category-structure/index.html`: 전체 카테고리 트리 + MD 판단 근거 + 적용 시 주의사항 포함

## 검증
- HTML 파일을 브라우저에서 열어 구조 확인
- 각 카테고리별 MD 판단 근거가 포함되어 있는지 확인
