전체상품 기본 정렬 변경

전체상품의 기본 정렬을 "인기도순"에서 "누적 판매순"으로 변경합니다.
스마트스토어 센터 ➜ 스토어 전시관리 ➜ 스토어홈 ➜ 전체상품 ➜ 기본 정렬
매출 포인트: "누적 판매순"은 리뷰가 많고 검증된 상품이 먼저 노출됩니다. 신규 방문자에게 "이 가게는 잘 팔리는 가게"라는 인식을 심어줍니다. 인기도순은 네이버 알고리즘 기반이라 통제가 어렵지만, 판매순은 실제 매출 실적 기반입니다.

신상품 섹션 정리

신상품에 계절 미스매치 상품이 섞여있지 않은지 확인합니다.
스마트스토어 센터 ➜ 스토어 전시관리 ➜ 스토어홈 ➜ 신상품
  1. 신상품은 보통 최근 등록순으로 자동 노출됩니다
  2. 동계 상품(기모 언더셔츠 등)이 신상품에 포함되어 있다면 해당 상품의 등록일을 확인
  3. 수동 선택 가능한 경우: 시즌에 맞는 상품(야구화, 배팅장갑, 보호장비 등)을 우선 배치
  4. 미즈노 HAGA 글러브(90만원)는 신상품에 유지 가능 - 프리미엄 이미지 형성
팁: 신상품에 노출할 상품이 부족하면 기존 인기 상품을 "재등록"해서 신상품 섹션에 올리는 방법도 있습니다. 단, 기존 리뷰가 사라지므로 리뷰가 적은 상품만 대상으로 합니다.
