5. 아코디언

FAQ, 상품 상세 정보 등 접기/펼치기 구현. max-height 애니메이션 방식 적용.

SSK 프로엣지 PEG144는 11.5인치 내야수용 글러브입니다. 특히 유격수(SS)와 2루수(2B) 포지션에 최적화되어 있으며, 빠른 공 처리가 필요한 내야 수비에 적합합니다. 포켓이 얕고 개폐가 부드러워 빠른 전환 송구에 유리합니다.

천연가죽 특성상 초기 형태 잡기(길들이기) 과정이 필요합니다. 구매 후 글러브 오일을 가볍게 도포한 후 공을 넣고 밴딩하는 방식을 권장합니다. 짱베이스볼에서는 별도 요청 시 기본 형태 잡기 서비스를 제공합니다.

수령 후 7일 이내 미사용 상품에 한해 교환/반품이 가능합니다. 단, 형태 잡기(오일 도포, 밴딩 등) 작업이 진행된 경우 소비자 사용으로 간주되어 교환/반품이 불가합니다. 자세한 사항은 고객센터(1588-0000)로 문의해 주세요.

function switchTab(btn, targetId) {
  btn.closest('.tabs').querySelectorAll('.tab-btn').forEach(b => b.classList.remove('is-active'));
  btn.classList.add('is-active');
  btn.closest('.comp-demo').querySelectorAll('.tab-panel').forEach(p => p.classList.remove('is-active'));
  document.getElementById(targetId).classList.add('is-active');
}
