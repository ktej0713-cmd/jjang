## 1. MOS란?

**미즈노 오리지널 시뮬레이션 (MIZUNO ORIGINAL SIMULATION)**
NPB 현역 선수 모델을 기반으로 8단계 커스터마이징을 거쳐 나만의 글러브를 제작하는 BSS 전용 주문 시스템.

| 항목 | 내용 |
|------|------|
| 대상 | BSS(Baseball Specialty Shop) 인증점 방문 고객 |
| 기본 가격 | ¥77,000 (본체 ¥70,000 + 세금 10%) |
| 납기 | 30~80일 (시기에 따라 변동) |
| 제조 | 일본 제조 (미즈노 테크닉스 하가 공장) |
| 모델 수 | 총 87종 (투수 32 / 내야수 24 / 외야수 15 / 포수 11 / 1루수 5) |

## 3. 주요 기술 사양 — 상세페이지 카피 활용

| 기술명 | 설명 | 상세페이지 활용 |

| **5Dベロ (5D벨로)** | 벨로부를 5차원 구조로 성형해 피팅감 향상 | "손등에 맞게 입체적으로 설계된 벨로" |

| **KMフロウ (KM플로우)** | 손가락 힘이 글러브로 전달되기 쉬운 배면 형상 | "힘을 자연스럽게 전달하는 배면 설계" |

| **小指2本ハミダシ** | 소지 두 개 넣기, 외야 글러브 안정성 향상 | "소지 두 개 넣기로 포켓 안정화" |

## 5. 상세페이지 제작 시 활용 가이드

### 상품 유형별 포지셔닝

| 상품 유형 | 타겟 고객 | 핵심 카피 방향 |
|---------|---------|-------------|
| MOS 특정 선수 모델 재고품 | "XX 선수처럼 치고 싶다"는 팬 심리 + 납기 없이 바로 받고 싶은 수요 | 선수명 + 모델 특징 + "납기 없이 바로 수령" |
| MOS 기본형 재고품 (내야-N, 외야-M 등) | 특정 선수 팬보다 스펙 중심 구매자 | 기술 사양(5D벨로·AXI 등) + 포켓 특성 + 가죽 등급 |
