# 구조 MD 메모리

## 짱베이스볼 카테고리 구조 현황

### 주요 문제점 (2026-03-01 분석)
1. 유소년 중복 분류: 001(글러브) + 003(배트) + 021(유소년용품) 3곳에 유소년 상품 분산
2. 포수장비 분리: 005(보호장비) + 044(포수장비) 이중 구조 → 통합 권장
3. 야구공 대분류 누락: 유소년용품(021) 내부에만 존재
4. 의류 세분화 부족: 트레이닝복/언더셔츠/방한의류 미분류
5. 훈련용품 대분류 없음: 피칭머신, 티볼, 네트 등 독립 대분류 필요

### 네이버 쇼핑 카테고리 매핑 기준
- 스포츠/레저 > 구기종목 > 야구 (최상위 경로)
- 카테고리명에 SEO 키워드 자연 포함 필수
- catId는 네이버 쇼핑 URL 파라미터에서 직접 확인

### SEO 핵심 키워드
- 글러브: "야구글러브", "사회인야구글러브", "포수미트"
- 배트: "야구배트", "나무배트", "알루미늄배트", "카본배트"
- 보호장비: "야구보호장비", "타자헬멧", "암가드"
- 의류: "야구유니폼", "야구바지", "야구모자"

## 카테고리 설계 패턴
- 3depth 이내 유지 원칙
- 포지션별 분류(투수/내야/외야/포수)는 글러브에서만 적용
- 타겟별 분류(유소년/사회인/엘리트)는 별도 진입점으로 제공 권장
