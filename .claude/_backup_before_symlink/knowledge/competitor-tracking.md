# 경쟁몰 모니터링

## 주요 경쟁사

### 1. 야구야 (yaguya.co.kr)
- 유형: 자사몰 + 네이버 스마트스토어
- 강점: 오랜 업력, 넓은 브랜드 라인업, 커뮤니티 인지도
- 약점: UI 구식, 콘텐츠 업데이트 느림
- 가격 전략: 중간가 유지, 쿠폰 위주 할인
- 주력 상품: 글러브, 배트 전 브랜드

### 2. 네이버 스마트스토어 (개인셀러)
- 유형: 스마트스토어 다수 경쟁
- 강점: 네이버 검색 노출, 쇼핑 라이브
- 약점: 전문성 부족, A/S 불안정
- 가격 전략: 최저가 경쟁, 박리다매
- 주력 상품: 소모품, 액세서리

### 3. 번개장터/중고나라
- 유형: 중고 거래 플랫폼
- 강점: 가격 경쟁력 (중고/미사용)
- 약점: 품질 보장 불가, 교환/반품 어려움
- 영향: 고가 글러브(20만+) 시장에서 경쟁

### 4. 야구넷 (yagooonet.com)
- 유형: 자사몰
- 강점: 일본 직수입 라인
- 약점: 소규모 운영, 배송 느림
- 주력 상품: 일본 직수입 글러브

## 차별화 전략 (짱베이스볼)
1. **전문성**: 야구를 아는 사람이 운영 → 콘텐츠 품질로 승부
2. **큐레이션**: 단순 나열이 아닌 상황별/포지션별 추천
3. **신뢰**: 과장 없는 실사용 기반 설명
4. **멀티채널**: 자사몰 + 네이버 스마트스토어 동시 운영
5. **타겟 세분화**: 사회인/유소년/엘리트 맞춤 접근
