# 시즌 체크

현재 날짜 기준으로 짱베이스볼이 집중해야 할 시즌 전략을 요약하는 커맨드입니다.

## 입력
인자 없이 실행하면 현재 날짜 기준 분석, 특정 월을 지정할 수도 있습니다.
- `/시즌체크` - 오늘 기준
- `/시즌체크 5월` - 5월 기준 분석

인자: $ARGUMENTS

## 실행 절차

### 1단계: 시즌 캘린더 로드
- `~/.claude/knowledge/seasonal-calendar.md` 읽기
- 현재 월(또는 지정 월) 기준 시즌 정보 추출

### 2단계: 시즌 브리핑 출력

#### 현재 시즌 요약
| 항목 | 내용 |
|------|------|
| 현재 시즌 | (개막/본시즌/하계/추계/비시즌) |
| 핵심 이벤트 | 진행 중인 대회/이벤트 |
| 주력 상품 | 이 시기 가장 수요 높은 상품군 |
| 핵심 타겟 | 사회인/유소년/엘리트 |
| 다음 시즌 예고 | 다음 달 준비 사항 |

#### 이번 달 프로모션 체크리스트
- 메인 배너 교체 여부
- 기획전 페이지 오픈 여부
- SEO 키워드 반영 여부
- 시즌 상품 노출 순서 조정 여부
- 네이버 스마트스토어 키워드 업데이트

#### 추천 액션
1. 지금 당장 해야 할 것 (긴급)
2. 이번 주 내 해야 할 것 (중요)
3. 이번 달 내 해야 할 것 (준비)

#### SEO/마케팅 키워드
- 메인 키워드 3개
- 롱테일 키워드 5개
- 네이버 쇼핑 상품명 반영 키워드

### 3단계: 지난달 vs 이번달 변화
- 새로 집중해야 할 카테고리
- 노출 줄여야 할 카테고리
- 가격/프로모션 조정 필요 상품

## 출력 원칙
- 한눈에 파악 가능한 요약
- 실행 가능한 액션 위주
- 짱베이스볼 운영에 바로 적용 가능
