# 상세페이지 미리보기

생성된 상세페이지 HTML 파일을 브라우저에서 미리보기합니다.

인자: $ARGUMENTS

---

## 실행 절차

### 1단계: 파일 결정

`$ARGUMENTS`가 있으면:
- 파일명 일부 또는 전체 경로로 해석
- `~/.claude/output/상세페이지/` 디렉토리에서 매칭되는 파일 검색

`$ARGUMENTS`가 없으면:
- `~/.claude/output/상세페이지/` 디렉토리에서 가장 최근 수정된 HTML 파일 선택

### 2단계: 파일 목록 표시

해당 디렉토리의 HTML 파일 목록을 수정일시 기준 최신순으로 표시한다 (최대 10개).
각 파일의 수정 시간과 파일 크기를 함께 보여준다.

### 3단계: 브라우저 열기

선택된 파일을 Bash 도구로 `start "파일경로"` 명령을 실행하여 기본 브라우저에서 연다.

**Windows 명령:**
```
start "" "C:/Users/jj1/.claude/output/상세페이지/파일명.html"
```

### 4단계: 결과 보고

- 열린 파일명과 경로를 표시
- 파일에 포함된 섹션 목록 (`<!-- [SEC:섹션명] -->` 마커 기준)을 간략히 표시
