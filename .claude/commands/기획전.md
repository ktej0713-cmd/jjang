# 기획전 생성 워크플로우

짱베이스볼 기획전을 설계하고 HTML 결과물까지 생성하는 원스톱 커맨드입니다.

## 입력
사용자가 기획전 주제를 제공합니다. 예시:
- `/기획전 3월 개막시즌`
- `/기획전 유소년 입문`
- `/기획전 글러브 포지션별`

인자: $ARGUMENTS

## 실행 절차

### 1단계: 시즌/타겟 분석
- `~/.claude/knowledge/seasonal-calendar.md`를 읽고 현재 시즌에 맞는지 확인
- 타겟(사회인/유소년/엘리트) 자동 판별
- 관련 프로모션 키워드 추출

### 2단계: 상품 구성
- `~/.claude/knowledge/brand-catalog.md`에서 관련 브랜드/상품 선정
- `~/.claude/knowledge/price-range-guide.md`에서 가격대별 라인업 구성
- 기획전 상품 10~20개 선정 (가격대 다양하게)

### 3단계: 기획전 설계
아래 표준 형식으로 출력:

| 항목 | 내용 |
|------|------|
| 기획전명 | SEO 키워드 포함, 매력적인 카피 |
| 타겟 | 사회인/유소년/엘리트 |
| 운영 기간 | 시작일~종료일 |
| 상품 구성 | 상품명/브랜드/가격/추천이유 표 |
| 할인 구조 | 쿠폰/적립금/세트할인 등 |
| 배너 카피 | 메인카피 + 서브카피 + CTA |
| KPI | 페이지뷰/전환율/매출 목표 |

### 4단계: HTML 생성
- `~/.claude/output/기획전/` 폴더에 HTML 파일 생성
- 파일명: `기획전_{주제}_{날짜}.html`
- 반응형 디자인, 짱베이스볼 브랜드 톤 적용
- 상품 카드 그리드 레이아웃
- 배너 영역 + 상품 리스트 + 혜택 안내 구조

### 5단계: 검증
- 야구인 관점에서 상품 구성이 자연스러운지 확인
- 과장 표현 없는지 확인
- 가격 정보 정확성 확인

## 출력 원칙
- 바로 쇼핑몰에 적용 가능한 완성도
- 과장 광고 표현 절대 금지
- "야구를 아는 사람이 기획한 기획전" 느낌 유지
- 네이버 스마트스토어 + 자사몰 동시 활용 고려
