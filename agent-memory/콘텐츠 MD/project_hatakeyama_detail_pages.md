## 확립된 표준 구조 (10섹션 — 글러브/미트용)
1. SEC01: 히어로 (배경 그라데이션, 오더 등급 배지, 1줄 정의)
2. SEC02: 필요 상황 / 이런 분께 맞습니다 (2x2 그리드, 재고 안내 포함)
3. SEC03: 핵심 스펙표 (좌열 헤더 #1B2A4A, 교차 배경)
4. SEC04: 소재/기술 심화 (가죽 + 백스타일 + 펠트 각각 별도 블록)
5. SEC05: 실제로 써보면 이렇습니다 (포구감/피팅감/조작성)
6. SEC06: 포지션/유형별 추천 (3열 카드, 적합도 등급 표시)
7. SEC07: 길들이기 가이드 (3가지 방식 비교표)
8. SEC08: 비교표 (이 상품 강조 #FFFBF0 배경 + #D4A843 헤더)
9. SEC09: FAQ (펼침 고정 방식 — details 태그 사용 불가)
10. SEC10: 추천 포인트 + 배송/교환/반품 안내

## 하타케야마 브랜드 핵심 기술 용어 정의
- **스페셜 레더**: 실버오더 이상 등급에 사용하는 오더 전용 특수 가죽. 기성품과 밀도·탄성이 다름.
- **쉘라무브(Shellarmove)**: 갑각류 관절 움직임에서 착안한 백스타일 독자 기술. 갑부 주름 구조로 미트 개폐 저항 감소, 포구 조작성 향상.
- **PRO-M2 패턴**: 얕은 포켓 설계. 빠른 송구 특화. 양의지(두산베어스) 등 KBO/NPB 프로 포수 사용 패턴.
- **실버오더**: 하타케야마 커스텀 오더 등급. 프로 스페셜 오더보다 합리적, 레귤러 기성품보다 상위.
- **길들이기 3종**: 기본(무료) / 수타(장인 직접 타법, 무료) / 물형부(형부 사용 정밀 포켓, 무료)

## 주의사항
- 가격 HTML 하드코딩 금지 (비교표에도 가격 행 제외)
- 길들이기 후 반품 불가 안내는 SEC07, SEC09(FAQ), SEC10(배송안내) 세 곳에 모두 포함
- 재고 1개 희소성 안내는 SEC02 하단 박스에 배치 (과장 표현 없이 사실만)
