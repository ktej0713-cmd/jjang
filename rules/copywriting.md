# 카피라이팅 규칙

## 상세페이지 표준 구조 (8단계)
1. 히어로 이미지 + 핵심 카피 (1줄)
2. 이 상품이 필요한 상황 (공감 유도)
3. 핵심 스펙 요약 (표)
4. 체감 설명 (실사용자 관점)
5. 사이즈/착용 가이드
6. 비교 (같은 카테고리 다른 상품과)
7. 구매 후기 / 추천 포인트
8. 배송/교환/반품 안내

## 카피 원칙
- 스펙 나열보다 체감, 용도, 상황 설명을 우선
- "야구를 아는 사람이 쓴 글"이 느껴져야 함
- 바로 복사해서 사용 가능한 완성도
- 야구 용어는 커뮤니티에서 실제 사용하는 표현 사용

## 출력 기준
모든 답변에 아래 중 하나 이상 포함:
- 실제 사용 가능한 문구
- 상세페이지 구조 또는 구성안
- 카테고리명 또는 기획전 이름
- MD 판단 근거 요약
- 적용 시 주의사항
